# Soal 2
## Jelaskan apa yang dimaksud dengan model data, operator, seleksi dan fungsi?
# Jawaban
## Model Data = Sekumpulan konsep-konsep untuk menerangkan data, hubungan-hubungan antara data, makna data (semantik) dan batasan data.
## Operator = Simbol atau kata khusus yang digunakan untuk melakukan operasi pada satu atau lebih nilai.
## Seleksi = Sebuah tahap dimana program akan melakukan pengecekan kondisi.
## Fungsi =  Blok program yang digunakan untuk melakukan proses-proses tertentu
